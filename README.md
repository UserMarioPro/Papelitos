## <a name="español"></a> 🇪🇸 Español

**Papelitos** es una plataforma revolucionaria que te permite dejar mensajes (notas) anclados a coordenadas GPS reales en cualquier parte del mundo. Los mensajes desaparecen del mapa global automáticamente después de un tiempo definido.

### 📥 Cómo descargar (Solo Android)
1. Ve a la pestaña de **[Releases](../../releases)** en este repositorio.
2. Descarga el archivo `papelitos.apk`.
3. Abre el archivo en tu dispositivo Android. Si el sistema te lo pide, acepta "Instalar aplicaciones de orígenes desconocidos".
4. ¡Inicia sesión como Invitado y empieza a poblar el mapa!

## <a name="english"></a> 🇬🇧 English

**Papelitos** is a revolutionary platform that allows you to leave messages (notes) anchored to real GPS coordinates anywhere in the world. Messages automatically disappear from the global map after a set time.

### 📥 How to Download (Android Only)
1. Go to the **[Releases](../../releases)** tab in this repository.
2. Download the `papelitos.apk` file.
3. Open the file on your Android device. If prompted, allow "Install apps from unknown sources".
4. Log in as a Guest and start populating the map!

## <a name="русский"></a> 🇷🇺 Русский

**Papelitos** — это революционная платформа, которая позволяет оставлять сообщения (заметки), привязанные к реальным GPS-координатам в любой точке мира. Сообщения автоматически исчезают с глобальной карты через заданное время.

### 📥 Как скачать (Только для Android)
1. Перейдите на вкладку **[Releases](../../releases)** в этом репозитории.
2. Скачайте файл `papelitos.apk`.
3. Откройте файл на вашем устройстве Android. При необходимости разрешите «Установку приложений из неизвестных источников».
4. Войдите как Гость и начните заполнять карту!
